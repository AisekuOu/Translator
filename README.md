# Translator

基于 OCR 和 翻译 API 

OCR API:

1.百度智能云OCR

2.腾讯AI开放平台OCR

3.Tesseract OCR

翻译 API:

1.百度翻译开放平台翻译

2.百度网页翻译

3.有道智云 · AI开放平台翻译

4.有道网页翻译

5.腾讯AI开放平台翻译
